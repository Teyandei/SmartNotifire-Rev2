# 📢 Smart Notifire

Smart Notifire is an app that announces notifications by text-to-speech (TTS).

## 🧠 Features
* Detects notifications based on user-defined conditions for the app name, channel, and title.
* Reads detected notifications aloud using the configured voice message.
* Lets you understand notifications by voice without checking the screen.

## 🔤 How to use

You can use this app in three steps:

1. Allow notification access
2. Add a rule
3. Receive a notification and hear the voice announcement
