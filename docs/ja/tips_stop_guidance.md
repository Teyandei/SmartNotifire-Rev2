# 🚫 音声案内をとめる方法

状況によっては、音声案内を止めたい場合もあります。  
Smart Notifireでは、いくつかの方法で簡単に調整できます。

## 📵 マナーモードにする

Smart Notifireは通知音に合わせて音声案内を行います。  
そのため、スマートフォンをマナーモードにすると音声案内も停止します。

ワンタップで切り替えられる、一番手軽な方法です。

## 🔚 ルールを無効化する

通知頻度の高いアプリは、音声案内も多くなります。  

必要に応じて、対象ルールをオフにしてください。  
通知音だけを残したい場合にも便利です。

## ⛔ アプリ通知をオフにする

Androidの通知設定から、対象アプリの通知自体をオフにする方法です。

通知そのものが不要な場合に利用できます。

## 📝 まとめ

スマートフォンの通知音だけでは、

- どのアプリの通知なのか
- どんな内容なのか

分からないことがあります。

Smart Notifireは、通知の意味を「音声で分かる」ようにするアプリです。

通知音が気になる場合でも、  
「どんな通知なのか」を知ったうえで、必要かどうか判断できるようになります。
