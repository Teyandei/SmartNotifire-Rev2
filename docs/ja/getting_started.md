# Getting Started

Smart Notifire(以下、本アプリ)は受けた通知が音声案内の対象になります。  
従って、インストール直後は通知を受けていない限り音声案内ルールを追加できません。  
本アプリは確認のため、あらかじめ音声案内ルールを一つ持っています。

<img alt="Basic Rule" src="https://teyandei.github.io/SmartNotifire-Rev2/images/ja/basic_rule.png" title="初期ルール" width="50%"/>
  
この音声案内ルールを使って本アプリの機能を確認してみましょう。  

> タップ操作のみで簡単に確認できます。
