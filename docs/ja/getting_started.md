# Getting Started

Smart Notifire(以下、本アプリ)は受けた通知が音声案内の対象になります。  
従って、インストール直後は通知を受けていない限り音声案内ルールを追加できません。  
本アプリは確認のため、あらかじめ音声案内ルールを一つ持っています。

<img alt="Basic Rule" src="https://teyandei.github.io/SmartNotifire-Rev2/images/ja/basic_rule.png" title="初期ルール" width="30%"/>
  
この音声案内ルールを使って本アプリの機能を確認してみましょう。  

> タップ操作のみで簡単に確認できます。
  
## 通知確認を行う

<img alt="Check Notification" src="https://teyandei.github.io/SmartNotifire-Rev2/images/ja/notification_check.png" title="通知確認画面" width="30%"/>

1. **通知**ボタンをタップすると本アプリがテスト通知を行います。
2. 通知の送信を許可するダイアログが表示されますので、**許可**をタップします。    
   <img alt="Allow Notification" src="https://teyandei.github.io/SmartNotifire-Rev2/images/ja/allow_notification.png" title="通知権限" width="30%"/>
3. 通知が送信され、通知音がなります。  

## 音声案内ルールを有効にする  
    
<img alt="Enable Rule" src="https://teyandei.github.io/SmartNotifire-Rev2/images/ja/enable_rule.png" title="ルール有効" width="30%"/>
  
- 許可スイッチをタップして音声案内ルールを有効にします。  

## 音声案内を確認する

- 再び、画面下部の**通知**ボタンをタップします。
- 通知音がなり、3秒後に音声メッセージを読み上げます。

> 以上が本アプリの動作です。次に音声案内[ルールの追加](./add_rule)をご覧ください。
