# Smart Notifire

Smart Notifire は、通知をテキスト読み上げ（TTS）で音声案内するアプリです。

## 特徴 
* 通知のアプリ名・チャンネル・タイトルに基づいて、ユーザーが設定した条件で通知を検出します。
* 検出した通知を、設定した音声メッセージで読み上げます。
* 通知内容を画面で確認しなくても、音声で把握できます。

## 使い方 

### [Getting Started](getting_started.html)



**利用手順:**
1. 設定画面で、対象とする通知条件と音声メッセージを登録します。
2. 条件に一致する通知が発生すると、設定した音声で案内されます。

## 技術仕様 (Technical Specifications)
本アプリはAndroid OSの最新バージョン（API Level XX以上）を対象としています。
安定して動作するために、「通知へのアクセス」および「音声合成サービス」への適切な権限が必要です。
