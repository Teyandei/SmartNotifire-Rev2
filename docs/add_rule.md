# ✏ Add a Voice Announcement Rule

This app announces received notifications by voice.  
 
## 1⃣ Open the add rule screen

<img src="./images/add_rule_button.png" title="Add Rule" alt="Add rule button" style="width:clamp(30%,calc(60000px - 10000vw),80vw);height:auto;max-width:100%;"/>

- Tap the add button on the screen.  

## 2⃣ Add a rule

- A list of notifications received so far is displayed.  

  <img src="./images/add_rule_screen.png" title="Add Rule Screen" alt="The screen after some time has passed since installation." style="width:clamp(30%,calc(60000px - 10000vw),80vw);height:auto;max-width:100%;"/>

- <img src="./icons/add_24.svg" width="16" style="vertical-align: middle;" alt="Add button"> Adds a rule for the selected row.  

## 📌 Display details

- The upper line shows the app name that sent the notification, and the line below it shows the channel name.
- **Received** is the number of notifications per day.
- <span style="background-color:#F9DEDC; color:red; padding:0 4px; border-radius:4px;">Configured</span> indicates a notification that already has a voice announcement rule.
- <span style="background-color:#6750A4; color:white; padding:0 4px; border-radius:4px;">Silent</span> indicates that the notification was a **silent notification** when it was received.
    > A **silent notification** is a notification that does not play a notification sound. This app does not announce silent notifications by voice.  
    > Silent notifications are usually configured by the app, but users can change the setting.
