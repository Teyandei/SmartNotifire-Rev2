## Privacy Policy

Smart Notifier (hereinafter referred to as "the App") respects user privacy and takes the protection of personal information with the utmost care. To provide its core function of reading notifications aloud via Text-to-Speech (TTS), the App accesses user notification data. However, **this data is never transmitted to any external server.**

### 1. Acquisition and Purpose of Personal and Sensitive Information

The App accesses and processes the following sensitive information only within the device:

-   **Notification Content (Notification Access)**  
    **[Purpose]** Used to read the title and body of notifications aloud based on user-defined rules and to display the "Notification Log."
-   **List of Installed Applications**  
    **[Purpose]** Used to allow users to select (filter) which applications should be subject to notification reading.
-   **Notification Title Cache**  
    **[Purpose]** Used to store the titles of previously received notifications to assist users when creating new reading rules.

### 2. Data Storage and Security (Network Communication)

-   **Offline Operation**: The App **does not request internet connection permission (INTERNET)**. Therefore, it is physically impossible for the acquired notification content or settings to be sent to the developer's or any third party's servers.
-   **Local Storage**: All data is stored only in a secure database (Room) within the device. When the App is uninstalled, all such data is completely removed.

### 3. Third-Party Provision

The App does not provide the acquired information to any third parties. Furthermore, it does not use any external SDKs for advertising or analytics.

### 4. Important Permissions Used

-   **Notification Access (Notification Listener Service)**  
    This is a mandatory permission to realize the core functions of the App (detecting and reading notifications).
-   **Post Notifications (POST_NOTIFICATIONS)**  
    Used to inform the user of the service's running status.

### 5. Data Deletion

Users can delete their history using the "Clear Notification Log" function within the App. Additionally, by uninstalling the App, all settings and cache data stored on the device can be completely deleted.

### 6. Changes to the Privacy Policy

This policy may be updated due to changes in Android specifications or the addition of App features. Changes become effective upon posting to this page.
